# Simple Interest Calculator

This project is a basic shell script that calculates simple interest based on user input.

## 📌 Description
The script takes three inputs:
- Principal amount
- Rate of interest
- Time period

It then calculates the simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## ⚙️ How to Run

1. Open terminal
2. Navigate to the project folder
3. Run the script using:

```bash
bash simple-interest.sh
